# 特許発明の検討ガイド

日本の特許法に基づいて、発明を考え、特許出願の可否を判断するための手順書です。

## 内容

- [発明とは](note.md#発明とは) - 特許法における発明の定義と種類
- [発明であるか](note.md#発明であるか) - 発明の成立要件（4要件）
- [記載要件](note.md#記載要件) - 出願書類と記載要件（5要件）
- [実体的要件](note.md#実体的要件) - 新規性・進歩性の判断基準
- [新規発明の検討手順](note.md#新規発明の検討手順) - 発明から出願までのフローチャート

## 使い方

### 発明のアイデアがある場合
1. [新規発明の検討手順](note.md#新規発明の検討手順) の Step 0 で課題・解決手段・構成要素を整理
2. フローチャートに従って先行技術調査と出願可否判断

### テーマが決まっていない場合
1. [パテントマップ](note.md#パテントマップ) で技術的な空白領域を探索
2. 課題を特定してから上記の手順へ
